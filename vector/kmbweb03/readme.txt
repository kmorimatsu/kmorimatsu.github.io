＜概要＞
KM-BASIC webは、KM-BASIC for MIPS用の簡易開発環境です。ブラウザ上でBASIC
プログラムの記述と実行シミュレーションによる動作確認を行なうことが出来ま
す。

＜使用方法＞
kmbasic.htmlをブラウザから開き、テキストボックスにBASICコードを記述した後、
「Run」ボタンを押して実行して下さい。

BASIC KM-1100
Compiling...done

と表示され、BASICプログラムが実行されます。BASICプログラムの動作確認が取れ
ましたら、*.txtファイルとして保存し、SDカードにコピーすれば、KM-BASIC for
MIPS上で実行することが可能です。

但し、シミュレーターの実行速度は、KM-BASIC for MIPSでの実行速度より劣りま
す。また、BGMや効果音の再生がとぎれとぎれになることがあります。加えて、こ
の開発環境でプログラムが実行可能であることは、必ずしもKM-BASIC for MIPSで
も実行可能であるとの保証にはなりません。文法等、KM-BASIC for MIPSのそれに
従って、記述して下さい。

ブラウザでhtmlファイルを開く際、kmbasic.html?debug=1とすれば、コンパイル
された中間言語の内容を閲覧することが出来ます。また、kmbasic.html?help=1と
すれば、BASICの仕様ファイルを閲覧しながら開発することが出来ます。

＜BASIC言語の書式＞
help.txtを参照して下さい。
