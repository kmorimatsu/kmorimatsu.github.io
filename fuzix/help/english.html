FZ/KM web is an HTML5 emulator of PC running Fuzix.

Following HTML5 functions are used:

  • Canvas
  • FileReader

To start FZ/KM, please access to the link, "fuzix.html" by browser. You will see a page as follows.

The black window is a display showing 80 x 24 characters. The indicator below is the current Z80 CPU speed (2 MHz is the default). "Set File" is used to upload a disk image file (will be explained below). "Help" is used to see this help page.

As display shows "Waiting for loading disk image", Fuzix cannot be started yet. To start Fuzix, a disk image file is required. The 128M bytes disk imaged for Tom's SBC is used. Please download "tomssbc-0.3.ide" (as noted on the display) from http://www.fuzix.org/ or somewhere else, then decompress and upload it by using "Set File" button. This file can be uploaded as a decompressed image as well as a ZIP archive. You will see as follows.

Please type "hda1" or "1" to the question, "bootdev:". Then, you will be asked to input the current date and time. If specifying is not required, plase press just enter key.

Please type "root" to the question, "login:", in the beginning. Now you can use Fuzix on your browser.

You can save the disk image file after using Fuzix. Please "shutdown" first on the Fuzix screen and wait until the OS will be terminated as follows.

Now, you can press "Save Disks" button to download the disk image as ZIP archive. This file can be directly used as a disk image by the "Set File" button in the next time.

The Fuzix ver 0.3 compiled for Tom's SBC is running here.

This HTML5 software emulates a PC like Tom's SBC (https://easyeda.com/peabody1929) and has following features.

  • CPU is Z80
  • 64 k bytes RAM is loaded
  • 64 k bytes ROM is loaded
  • Upon the reset, ROM is used in the address region of 0000h-3FFFh
  • Writing to I/O port address 38h defines which ROM or RAM is used in the address 0000h-3FFFh (0:ROM; 1:RAM)
  • Writing to I/O port addresses 3Eh and 3Fh defines which bank is used for ROM (3Eh: A15; 3Fh: A14)
  • Z80-SIO is connected to I/O port addresses 00h-03h for output to the display and input from the key board
  • A conpact flash card is connected to I/O port addresses 10h-17h for emulating a disk
You can add option(s) to URL when starting FZ/KM web. To use Z80 debug tools, please access to "fuzix.html?debug=1". To use different CPU speed from default, please access to, for example "fuzix.html?speed=8000000" in the case of 8 MHz clock speed.
